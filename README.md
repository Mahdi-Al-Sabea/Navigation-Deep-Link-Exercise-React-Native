## Run the app

1. Clone the repo
   ```bash
   git clone https://github.com/Mahdi-Al-Sabea/Navigation-Deep-Link-Exercise-React-Native.git
   cd Navigation-Deep-Link-Exercise-React-Native
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Run on Android
   ```bash
   npx react-native run-android
   ```
